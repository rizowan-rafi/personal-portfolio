## 👋 Introduction
This portfolio is built with **Next.js 15** and **React 19**, representing the cutting edge of web development. It features a sleek, dark-themed UI with fluid animations, responsive layouts, and a seamless user experience designed to showcase my engineering skills and academic projects at **RUET**.

---

## ✨ Features & Navigation
The application is structured into five core sections designed for a seamless user journey:

* **🏠 Home:** High-impact hero section featuring a dynamic introduction and a direct "Download CV" call-to-action.
* **📂 Projects:** A curated showcase of full-stack applications (like **MedMarket** and **GigFlow**) featuring tech-stack badges and integrated GitHub links.
* **📄 Resume:** An interactive, tab-based interface showcasing **Experience**, **Education**, and **Skills** using Radix UI for maximum accessibility.
* **📞 Contact:** A functional lead-generation form with real-time validation and localized contact information.
* **💼 Hire Me:** A strategic conversion button that instantly redirects users to the contact interface to initiate collaboration.

---

## 🛠️ Technical Stacks
Built with the latest technologies to ensure speed, SEO optimization, and fluid animations.

### Core Frameworks
* **Next.js 15 (App Router):** Leveraging Server-Side Rendering (SSR) and optimized routing.
* **React 19:** Utilizing the latest concurrent rendering features for a snappier UI.
* **Tailwind CSS:** A utility-first approach for pixel-perfect, responsive styling.

### UI & UX Enhancements
* **Framer Motion:** Powering high-end micro-interactions and scroll-reveal animations.
* **Radix UI:** Headless primitives used for accessible Tabs, Dialogs, and Tooltips.
* **Swiper.js:** Implementation of modern, touch-friendly project sliders.
* **Lucide React & React Icons:** Consistent, high-quality iconography across all routes.

## 🚀 Getting Started

####  Clone the repository:
```bash
git clone https://github.com/rizowan-rafi/personal-portfolio.git
cd personal-portfolio
```

#### Install dependencies:
```bash
npm install
```

##### Configure Environment:
```bash
Create a .env.local file to include your EmailJS and Resend API keys.
```

### Run Development Server:
```bash
npm run dev
```
